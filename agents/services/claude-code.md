# Claude Code

このファイルは Claude Code 用の補足メモです。
共通ルールは `agents/base.md`、プロジェクト固有ルールは `agents/projects/<project>/AGENTS.md` を使います。
`CLAUDE.md` は Claude Code の入口です。

## Notes
- Claude Code の読み方、前提、好ましい作業の流れだけを書く。
- `CLAUDE.md` から読まれる前提で、ここは差分だけに絞る。
- Codex 固有の runtime state や `codex/` 配下の説明は書かない。
- 共通化できるものは `agents/base.md` に寄せる。
